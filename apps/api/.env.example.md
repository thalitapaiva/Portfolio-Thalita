# Environment variables

This app reads its configuration from the **repository root `.env`** file.

See `../../.env.example` at the monorepo root for the canonical list of
variables. Copy it to `.env` in the repo root and edit as needed:

```bash
cp .env.example .env
```

There is no per-app `.env` file for `apps/api` — everything is loaded via
`@nestjs/config` from the root `.env` (see `src/app.module.ts`).
